# Frontend Mentor - Solução do FAQ Accordion

Este é um projeto de FAQ Accordion desenvolvido como parte de um desafio do Frontend Mentor.
O objetivo foi criar uma interface simples, responsiva e acessível, utilizando HTML semântico e CSS moderno.

## Visão geral

O projeto foi desenvolvido com o objetivo de criar uma interface interativa e responsiva para exibir perguntas frequentes, onde os usuários podem expandir e recolher as respostas de forma intuitiva.

## Funcionalidades

- Interação de abrir/fechar ao clicar nas perguntas
- Implementação semântica utilizando `<details>` e `<summary>`
- Design responsivo com abordagem mobile-first
- Estados visuais para hover
- Acessibilidade aprimorada

- CSS3
- Flexbox
- Abordagem Mobile-First
- Metodologia BEM para organização CSS

## O que aprendi

Durante o desenvolvimento deste projeto, pude aprimorar meus conhecimentos em:

- Aplicação de HTML semântico para melhorar acessibilidade e estrutura
- Utilização de `<details>` e `<summary>` como solução nativa para accordions
- Implementação de responsividade fluida usando `clamp()`
- Organização escalável de CSS seguindo a metodologia **BEM**
- Criação de interfaces responsivas com foco na experiência do usuário
